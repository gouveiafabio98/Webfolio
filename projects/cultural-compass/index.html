Cultural Compass

Cultural Compass, inspired by the nautical instruments found in the Chímico Laboratory, aims to guide the user in exploring various cultural points of Coimbra using a physical compass that provides digital instructions in the physical world.
Academic Project — MDM 2022 Tools: Blender, A-Frame, HTML, CSS, JS
